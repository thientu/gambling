# Gambling Web App

A modern web application built with Vite + vanilla JavaScript, deployed via GitHub Pages.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project uses GitHub Actions for automatic deployment to GitHub Pages. To enable:

1. Go to repository **Settings**
2. Navigate to **Pages** (under "Code and automation")
3. Set **Source** to `GitHub Actions`
4. Save changes

The app will be deployed at: `https://thientu.github.io/gambling/`

## Tech Stack

- **Vite** - Build tool and dev server
- **Vanilla JavaScript** - No framework
- **GitHub Pages** - Static hosting
- **GitHub Actions** - CI/CD deployment
